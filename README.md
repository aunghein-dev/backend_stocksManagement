# 📦 Backend - Stocks Management System

This is a Spring Boot REST API for managing stocks, checkouts, users, and dashboards.

## 🚀 Tech Stack

- Java 17
- Spring Boot 3
- Spring Security (JWT)
- Supabase Integration
- Docker & Docker Compose
